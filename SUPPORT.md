# Nida — Support

This repository hosts **remote content** (JSON, images, `manifest.json`) for the **Nida** app, delivered over the CDN described in the [README](README.md). For **app** issues (prayer times, notifications, Qibla, device behaviour), use the contact below.

The Nida app UI is available in: **English, Turkish, Arabic, Portuguese, Spanish, German, Dutch, Kurdish (Kurmanji), Kurdish (Sorani).** This page repeats the same support information in those languages.

**App Store Support URL:**  
`https://github.com/ismile94/nidaunnur-content/blob/main/SUPPORT.md`

## English

### Contact

- **Email:** [ismailaram94@gmail.com](mailto:ismailaram94@gmail.com)

Please include your **device model**, **OS version** (e.g. iOS 18), and **app version** when you write in.

### What we can help with

- **Nida app:** prayer times, location, notifications, Quran playback, Qibla / maps, crashes, or anything you see inside the app.
- **Remote content in this repo:** wrong or missing seasonal images/text, `manifest.json` date ranges, or CDN delays after a recent publish.

### Common quick checks (app)

1. **Prayer times look wrong** — Check **location permission** and the selected city; try manual location if GPS is off.
2. **No notifications** — **Settings → Nida → Notifications** on iOS; check **Focus / Do Not Disturb** and in-app notification settings.
3. **Qibla / compass unstable** — Move away from metal and speakers; calibrate the compass if the system suggests it.
4. **Maps / nearby** — Location on; internet required for map data.

### Privacy

Use the **Privacy Policy** link on the App Store product page when available, or ask by email.
